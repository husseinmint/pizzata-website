# Gallery Images

Add your food images to the `public` folder with the following names:

- `gallery-1.jpg` - Green beverage image
- `gallery-2.jpg` - Ramen bowl image
- `gallery-3.jpg` - Dumplings image
- `gallery-4.jpg` - Sushi rolls image
- `gallery-5.jpg` - Salmon nigiri image
- `gallery-6.jpg` - Soup bowl image

**Recommended image specifications:**
- Format: JPG or PNG
- Dimensions: 389x466px (or similar aspect ratio)
- Optimized for web

If images are not provided, placeholder images will be displayed automatically.
